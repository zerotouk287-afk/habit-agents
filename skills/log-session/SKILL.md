# SKILL: log-session

## 트리거
"~시간 했어", "~분 공부함", "/track" 등 세션 기록 의도가 보이는 모든 입력

## 절차
1. 입력에서 추출: habit(dev/realestate), 날짜(기본 오늘), 분 단위 시간, 메모, 링크
2. 모호한 항목이 있으면 **한 번에 한 질문만** (예: "몇 분 정도였어요?")
3. `data/logs/YYYY-MM.jsonl` (해당 월 파일, 없으면 생성)에 한 줄 append:
   `{"ts":"...","habit":"...","minutes":N,"note":"...","links":[...]}`
4. **노션 동기화**: 해당 habit의 goals.json `notion_db` 데이터베이스에 행 1개 생성
   (이름=note, 날짜, 분). 성공 시 로그 항목 "synced": true + 페이지 URL을 links에 추가,
   실패 시 "synced": false (에러로 멈추지 말 것)
5. append 직후 해당 habit의 이번 주 누적/목표/달성률을 다시 계산해 한 줄로 보고.
   노션 동기화 성공 여부도 포함: 「✅ 기록 + 노션 동기화 완료 — 이번 주 🎹 1.5h / 2.5h (60%)」
6. 대시보드는 라이브 페이지라 재생성 불필요 (자동 반영됨)

## 검증
- minutes는 1~960 범위. 벗어나면 오타로 보고 확인 질문
- 같은 날 같은 habit 중복 기록은 허용 (세션이 여러 번일 수 있음)
